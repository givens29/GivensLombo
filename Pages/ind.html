Delivery International

Delivery International adalah sebuah website pemesanan makanan yang memungkinkan pengguna untuk mendaftar, mencari menu, memesan makanan, memberikan rating, dan lainnya. ... Teknologi yang digunakan untuk mengembangkan sistem ini adalah React untuk frontend, ASP .NET Core untuk backend, serta SQL Server untuk menyimpan, mengambil, dan mengelola basis data.

MOKERJA

MOKERJA adalah sebuah website lowongan pekerjaan yang membantu menghubungkan pencari kerja di Papua dengan perusahaan yang sedang mencari karyawan asal Papua. ... Website ini memungkinkan pencari kerja untuk mendaftar (sistem memiliki mekanisme untuk memverifikasi apakah pencari kerja berasal dari Papua atau bukan), mengelola profil, memverifikasi akun, mencari lowongan pekerjaan, mencari perusahaan, mengirimkan lamaran kerja, dan lain-lain.Perusahaan juga dapat mendaftar, mengelola profil, menambahkan perekrut (recruiter), serta memasang lowongan pekerjaan (untuk dapat memasang lowongan, perusahaan harus berlangganan ke MOKERJA), dan mengelola data lowongan. Teknologi yang digunakan adalah React, ASP .Net Core, SQL Server, Midtrans and Cloudinary.

SMS

Scholarship Management System (SMS) adalah sebuah website yang membantu menghubungkan penerima beasiswa Otonomi Khusus Papua dengan pengelola beasiswa. ... Website ini memungkinkan penerima beasiswa untuk mendaftar dengan mengunggah kontrak beasiswa mereka, yang kemudian akan diverifikasi oleh pengelola beasiswa sebelum diberikan izin untuk login.Penerima beasiswa juga dapat mengunggah catatan akademik serta laporan pengeluaran untuk memproses beasiswa mereka secara berkelanjutan. Sementara itu, pengelola beasiswa dapat memverifikasi penerima, serta meninjau catatan akademik dan pengeluaran mereka untuk memproses beasiswa. Website ini juga menyediakan fitur pesan yang memungkinkan penerima dan pengelola beasiswa berkomunikasi langsung. Website ini dikembangkan menggunakan HTML, CSS, JavaScript, ASP .Net Core and SQL Server.
